

# DIX PAS — Spécification V1 (Version Performance)

## 🎛️ Concept

**DIX PAS** est un **step-sequencer MIDI monophonique orienté performance live** :

- 1 séquence de **10 pas**
- interface compacte et directe
- contrôle par **boutons + encodeur + LEDs + écran**
- variations musicales via :
  - **gamme globale**
  - **probabilité**
  - **ratchet (via shift)**
  - **sens de lecture**
- synchronisation MIDI :
  - clock interne
  - clock externe

---

## ⚙️ 1. Architecture générale

### Modes de clock

- **Interne** : génère tempo + envoie clock
- **Externe** : suit une machine (MPC)

## 🔁 3. Sens de lecture

Paramètre global :

```cpp
enum PlayMode {
  FORWARD,
  REVERSE,
  PINGPONG,
  RANDOM
};
```

### Comportements

- **FORWARD** : 0 → 9
- **REVERSE** : 9 → 0
- **PINGPONG** : aller-retour
- **RANDOM** : pas aléatoire à chaque step

👉 Très important pour le live et la variation.

## 🎛️ 7. Interface physique

### Rangée 1 (principale)
- 10 boutons = sélection des pas
- 10 LEDs

### Rangée 2 (contrôle)
- 10 boutons = PROBABILITÉ
- + 1 bouton SHIFT global

---

## 🔄 8. Logique des modes (très important)

### Mode NOTE (par défaut)
- encodeur → change la note
- boutons rangée 1 → sélection

### Mode PROBABILITÉ
- boutons rangée 2 → valeur de probabilité rapide
- encodeur → réglage fin

### Mode RATCHET (via SHIFT)
- maintenir SHIFT
- boutons rangée 2 → définissent le ratchet
- encodeur → réglage fin

👉 SHIFT transforme la rangée PROB en rangée RATCHET

---

## 🧠 9. Pourquoi cette architecture est excellente

- moins de boutons → plus lisible
- gestuelle simple
- SHIFT = accès rapide sans surcharge
- machine compacte
- parfaite pour performance live

---

## 🖥️ 10. Écran

- retour auto après inactivité

```text
Value: 75%
```
